## Implemented Metrics

The repository includes queries that calculate and analyze key business metrics, such as:

* **Average Order Value (AOV)**
  Calculated using order-level aggregation to reflect real business definitions

* **Revenue by Month**
  Time-based aggregation to analyze sales trends

* **Revenue by Category**
  Multi-table joins to evaluate category performance

* **Revenue by Author**
  Revenue attribution across related entities

All metrics are derived from relational joins across orders, order items, products, authors, and categories, reinforcing a **data modeling and analytical mindset**.
